# 学習用用 DB 作成スクリプトについて

## いつ使うか

本プロジェクトを始めて起動するとき、または Docker をきれいさっぱりしたときに実行してください。

## 使い方

visual studio にて docker を起動させた後に、init.ps1 を起動してください。

## メンテナンスは？

基本的にはこれで完成しているため、メンテナンスはしない予定です。
ただし、現時点での DB 定義では容量が足りないなどの問題があった場合は書き換えます。

## テーブル構築や、データについては？

visual studio のスキーマ比較やデータ比較機能を用いて、各人のデータを同期していくようにします。
